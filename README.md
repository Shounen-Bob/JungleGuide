チャンピオンを選んだらすること

1：ジャングルで勝つための基本原則
0.ジャングルはカメラとtabゲー
カメラを使わない限り、ガンクもダイブもできない
キャンプにスキルを使ったらふぁ
1. 勝利条件を把握する
基本的にJGの勝ち方は以下が大目標である。

ドラゴンを4つ取る
アタカンをとる
バロンを取ってシージする（インヒビターよりバロン・アタカンを優先すること）
スケーリングして集団戦で破壊し、オブジェクトをとる（自分のキャラに依存する）

★オブジェクト優先主義を徹底する（キルよりもオブジェクト）
キルを追いかけるのをやめ、常にオブジェクト（タワー、ドラゴン、ヘラルド、グラブ、アカタン、バロン）を軸に動く。
ここに関連しない小競り合いには参加しない方がいい。
2. 超重要：ジャングルのリズム（テンポ）を保つ
行動パターンは常に [ファーム] → [ガンク/オブジェクト] → [リコール] の繰り返しであるべき。
大原則として**「フルクリア → 自由時間 → リコール → ジャングルモンスターが再び沸く瞬間にそこに到着し狩る」**ことを意識する。リスポンしてからリコールして向かうのでは遅すぎる。
多くの低レートプレイヤーは、キルやファームの後にリコールせず、小競り合いに参加したりなどダラダラとマップに居座る。これでは装備更新ができず、次のアクションが遅れる。
仮にミッドのブッシュで30秒~1分も粘ってガンクして決めたとしても、自分は本来その間に3つのクリープをファームでき、そのキャンプのリスポンも早まり経験値が上がっていた。つまりノーリスクでさほど変わらないリターンを得られたことになる。
ファームする時間を大幅に使ってガンクするのは無駄な時間といえる。
1キルで得られる経験値は少なく、3キャンプ狩る方が経験値的にもゴールド的にも価値が高い
ほんとに大事な視点として、**「リコール → ジャングルモンスターが再び沸く瞬間にそこに到着する」**ことが全て。
3. 超重要：ストロングサイドとウィークサイド
★勝っているレーン（ストロングサイド）を中心にプレイする
理想はボットレーンがストロングサイドになること。
二人分勝たせられ、さらにADCが育てば勝利条件であるオブジェクトが取りやすくなるため。
ボットレーンに向かってフルクリアするのが基本（上から下へ）。低レートのボットレーンは荒れやすく、ダブルキルのチャンスが多い
アクション（ガンクやドラゴン）は必ずストロングサイドで行う。


★負けているレーン（ウィークサイド）には絶対に介入しない
負けているレーン（0/6/0の味方など）は助けなくていい。
彼らのために時間を使うより、勝っているレーンをさらに勝たせる方が効率的。
負けているレーン（ウィークサイド）側のジャングルキャンプはできるだけ早く狩り終えその場を離れること。事故に巻き込まれるしいいことがない。
「ウィークサイドのキャンプを先に狩り、ストロングサイドに向かう」。
4. 超重要：勝ちチームの動きと負けチームの動き
★勝ってるチームはオブジェクトを利用し5v5に持ち込む
相手が無理して来たらACEかつオブジェクトがとれ試合が終わることが多い。 

★負けてるチームは、ひたすらマップの逆でオブジェクトやファーム交換する
相手がたとえばドラゴンに集まっているのならそのドラゴンを捨て、TOP 1stタワーや相手のTOPジャングルを食う。これを徹底すること。
「代わりの行動」を示す: 味方が自殺しそうなら「危険」ピンを出した直後に、「ボットタワーを狙おう」「ドラゴンに行こう」と次の目標にピンを出すべき。そうしないと味方は突っ込んで死ぬ。


2：序盤のジャングラーの仕事と動き（～14分）
1. フルクリアを崩すべきタイミング
結論から言うと、「なんとなく」や「味方が呼んでいるから」で崩すのは厳禁。
以下の明確なリターンが確定している時のみ、AFKファームを解除。

オブジェクト（ドラゴン・ヘラルド・グラブ）が取れる時
「確実な」ダイブが見えている時（ダイブはGankの中でも一番価値がある）
確実なダイブの条件
容易にキルラインが見えている
相手のJGやローム、TPの介入がないとわかっている
味方が確実にダイブされそうな時（序盤の味方タワーは強く、ダイブへのカウンター成功率はかなり高い）
2. ファームのテンポを保つ
超重要：ジャングルのリズム（テンポ）を保つの項を徹底する。
特に味方のピンに惑わされテンポを崩さないようにする。
「1回アクション（ガンクなど）を起こしたら、最低でも自陣の片側サイド（3キャンプ）をクリアする」ことを基本リズムとする。
3. 装備購入とドラゴンの鉄則
★ファームが加速する装備から買う

前述の通り、ファームの速度や効率がJGの勝率に直結する。
ルビークリスタルやブーツなどは序盤絶対に買わない。
APならモンスターへのDMGボーナスがある「運命の灰」を優先。

★序盤のドラゴンを無理にソロで取らない

装備がない段階でのドラゴンは時間やHPを多く失ってしまうし、ドラゴン1体で発揮する効果も序盤はかなり薄い。
それが逆キャンプ3つを失うきっかけになると本末転倒になる。

★ムキにならず少しでもリスクがあるなら、オブジェクトは全て相手に渡す

たとえばbotがgankされてドラゴンをさわられたら、無理にコンテストはしない。
負けているサイド（ボット）には近づかず、手薄になった反対側の敵陣のリソースを奪うことで、全体的なゴールド差を広げさせないようにする。

★敵JGの位置を常に予測し、味方にピンを鳴らす

たとえば3:30でカニに到達したものの、敵JGがその場に現れなければ、十中八九逆サイドにいるので、pingを鳴らす。
4. ガンクの技術
★大前提として「確実なキル」以外はガンクしない

「フラッシュを落とさせるだけのガンク」は正直無意味。
ガンクするレーンは1つに絞るのが理想。
序盤ガンクをして良いのは1~2レーンで、1回のリコールまでに何度もいろんなレーンに走って行ったりガンクをしない。
まず1つのレーンから丁寧に潰していき、しっかりと装備を整えつつファーム速度を加速させ、キルラインが大きく広がってきたり動ける時間が増えてから他のレーンも視野に入れる。

★具体的なテクニック

体力100%の敵メレーをgankして殺すのは相当難しい：相手がタンクだと尚更。こういった場合は無理にガンクしない。
ガンクに行く際のピンは直前に鳴らす：低レートの味方は、ピンを見ると条件反射で前に出てしまう。自分の姿が敵に見える直前、あるいはCCを入れる直前にピンを鳴らす。
ジャングルを狩るときに、隣接レーンのウェーブ状況を確認する：たとえば青側で狼を狩ってる時はtop/midレーンを見て、ウェーブが押し込まれそうか、押しそうか、ダイブ出来そうか、されそうかを見る。このタイミングで見れれば、無駄に寄せたりする工数を減らせる。
フラッシュ/ブリンクを追いかけるスキルを最初に使わない：シンジャオを例に出すと、距離を詰める要のEは最初に使わない。ヘカリムR/レクサイE/マルファイトRなども相手のフラッシュやブリンクを見てから使う。
常に相手がミニオンをラストヒットする瞬間にガンクする：レーン戦と同じ概念。この瞬間相手は前に出過ぎることが多い。
敵がギリギリの体力にも関わらず、プッシュしきらないといけない場合を狙う：プライドファイトの後はよくウェーブ観察する。味方のウェーブの位置を見て、もう1ウェーブ押さないといけない状況であれば狙う。


3：中盤のジャングラーの仕事と動き（14分～）
1. 超重要：自分だけが育った時
キルを持っている（育っている）ジャングラーがやるべき最も重要なことは、レーンに介入してタワーを折る、または近くのオブジェクトをとること。
チームのキルのほとんどを自分が持っている場合、「自分がデスしたら負け」という意識を持つ。フラッシュやウルトがない時は無理をせずファームを優先する。
フラッシュは攻めではなく回避に使う。
負けてるチームは、ひたすらマップの逆でオブジェクトやファーム交換する原則を守る。相手の一番育ったやつをひたすら避け、逆サイドでのアクションを狙う。
自分だけが育ったら、集団戦では姿を隠す。
自分がキャリー枠（ダメージ源）である以上、CC（行動妨害）を受けて即死することは許されない。
自分が落ちたら終わりなので、集団戦では絶対にフォーカスを受けない位置取りを徹底する。
たとえタンクでも、自分が死んだらその集団戦は味方が皆殺しされるだけ。
敵が見えない場所には入らない。
味方が揃っていない集団戦は避ける。
「お金を持ったまま死ぬ」のが最悪なので、リコールして装備を最強状態に保つ。
自分が育っている中盤以降、集団戦で勝てなさそうな場合や暇なときは積極的に孤立する敵を狙いに行くと良い。
相手の5人の中で一番狙いやすいのは孤立してレーンを押している相手なので、ミニオン状況や位置などから誰がどこにファームにくるかなどを事前に読み切って向かうと決まりやすい
2. 自分が敵JGよりもかなり育ったら敵とルートを合わせる
自分が有利な時は、あえて対面と同じルートで動く。
そうすることで相手のアクションに対してイタイしっぺ返しをくらわせられる。

★育っている時、わざわざ何かを交換されるタイミングでドラゴンを急いで狩る必要はない

勝っているチームは交換されることを防ぐのが重要。
ドラゴンなどはアナウンスもあるので、TOPのキャンプやオブジェクトを交換されるなら、狩らない。
3. 中盤以降の鉄則：視界とミッドプッシュ
中盤以降の勝敗は「どちらが先にミッドレーンを押せるか」にかかっている。
先に押せば、相手はミニオン処理のためにレーンに縛られる。
その間に自分たちは自由に動け、視界を取ったり、孤立した敵をキャッチしたりできる。
逆に相手に先に押されると、視界を取りに行く際にリスクを負うことになる。
4. 具体的な中盤以降のオブジェクト準備
1分前からウルトを温存する：ドラゴンファイトの前にウルトを使ってしまうと、集団戦で勝てなくなる。
30秒前～1分前：オブジェクト（ドラゴンなど）の逆サイドのファームは捨てる。
視界確保：チームと共に視界を取り、相手の視界を消す。
ミッドプッシュ：ミニオンウェーブが来るタイミングに合わせてミッドに寄り、素早く処理する。
ライン上げ：視界とウェーブ状況の有利を活かして前進し、相手を圧迫する。相手が下がればオブジェクトを触る。
特に敵サポの動きを見る。敵サポが通った道はワードが置かれている。
5. 中盤以降はリコールタイミングを同期する
タワーを折った後など、味方がリコールするタイミングではジャングラーも一緒にリコールするべき。
自分だけ残ってファームを続けると、装備更新が遅れ、次のアクションで味方と足並みが揃わなくなる。また、孤立してキャッチされるリスクも増える。
ただし、基本は「次にアクションを起こしたいサイド（例えばバロン側）」に合わせてリコールする。
6. 集団戦前にTABキーを押して、誰が主人公か確認する
味方のジンクスが育っているなら：敵のバックラインに飛び込むのではなく、ジンクスを狙う敵を止めるのが正解。
味方のイレリアが育っているなら：一緒に敵のバックラインに飛び込み、フォーカスを分散させ暴れてもらうのが正解。
自分だけが育っているなら：上の項を参照。こういう時はあまり戦いを挑まない方がいい。
